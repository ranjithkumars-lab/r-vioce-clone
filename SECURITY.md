# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within **Voice Studio**, please send an email to security@voicestudio.ai instead of filing a public issue. All security vulnerabilities will be addressed promptly.
